# add-two-numbers

A simple Python utility that demonstrates two arithmetic operations.

---

## Functions

| Function | Description |
|---|---|
| `add_two_numbers(a, b)` | Returns the sum of two numbers |
| `multiply_two_numbers(a, b)` | Returns the product of two numbers |

---

## Usage

```bash
python add_two_numbers.py
```

**Example output:**
```
Enter first number: 5
Enter second number: 3
Addition:       5.0 + 3.0 = 8.0
Multiplication: 5.0 x 3.0 = 15.0
```
